# Customization

Branding is controlled from:

- `.env` for deployment defaults.
- `app_settings` for runtime client settings.
- Developer setup wizard at `/admin/setup`.

Main variables:

- `VITE_BRAND_NAME`
- `VITE_PRIMARY_COLOR`
- `VITE_SUPPORT_PHONE`
- `VITE_SUPPORT_WHATSAPP`
- `VITE_CURRENCY`
- `VITE_CURRENCY_SYMBOL`

Logo, login image, colors, support numbers, and company text should be changed per client before delivery.
