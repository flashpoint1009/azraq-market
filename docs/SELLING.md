# Selling Guide

Suggested packaging:

- One-time license for a single client.
- Monthly managed SaaS.
- Custom deployment package.

Before selling:

- Replace branding from the setup wizard.
- Configure Supabase and storage.
- Run all migrations.
- Test customer, admin, warehouse, and delivery flows.
- Generate a signed Android build if required.

Pricing is intentionally not hardcoded. Configure it per client after final testing.
